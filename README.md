# ✍️ Handwritten Character Recognition App

## 🤖 Overview
This is a Streamlit-based deep learning project that recognizes handwritten alphabets (A-Z) using a CNN model trained on EMNIST dataset.

---

## 🚀 Features
- Upload handwritten image
- Predict alphabet (A-Z)
- Real-time preprocessing
- CNN deep learning model
- Interactive Streamlit UI

---

## 🛠 Tech Stack
- Python
- TensorFlow / Keras
- OpenCV
- NumPy
- Streamlit
